# CompTIA Network+ N10-009 Study Guide

A comprehensive study guide for the CompTIA Network+ (N10-009) certification exam, synthesized from course materials and official exam objectives. Built as a single dark-mode HTML file with full anchor navigation.

🔗 **[View the Study Guide](https://conwilso92.github.io/network-plus-study-guide)**

## Coverage

All 16 topic areas mapped to the 5 official exam domains:

| Domain | Weight | Topics |
|---|---|---|
| Networking Concepts | 24% | Parts 1, 3, 4, 5 |
| Network Implementation | 19% | Parts 2, 6, 7, 8, 9, 16 |
| Network Operations | 16% | Parts 10, 11, 12 |
| Network Security | 19% | Part 13 |
| Network Troubleshooting | 22% | Part 14, 15 |

### What's Inside

- **Part 1 — Networking Fundamentals:** Network types (PAN/LAN/MAN/WAN/SAN), peer-to-peer vs client/server, OSI model (all 7 layers with PDUs and protocols), TCP vs UDP, TCP 3-way handshake, duplex modes, CSMA/CD vs CSMA/CA, broadcast and collision domains
- **Part 2 — Cabling & Connectors:** UTP vs STP, cable categories (Cat 5–6a), T568A/B wiring standards, Ethernet standards (100Base-TX through 10GBase-ER), fiber types (SMF/MMF), fiber connectors (LC/SC/ST/MTRJ), fiber polish types (PC/UPC/APC), transceivers (SFP/SFP+/QSFP), cable issues (attenuation, crosstalk, EMI)
- **Part 3 — Networking Devices:** OSI layer reference for all devices, DNS (record types, DNSSEC, DoH, DoT), DHCP (DORA process, relay, reservations, IPAM), IDS vs IPS, firewall types, proxy servers, load balancers
- **Part 4 — IP Addressing:** IPv4 address classes, special ranges (private, loopback, APIPA, broadcast), unicast/broadcast/multicast/anycast, NAT types (static, dynamic, PAT), IPv6 (format, shortening rules, SLAAC, link-local, global unicast)
- **Part 5 — Subnetting:** Full /8 through /32 reference chart, magic number method with two worked examples, binary reference table, octet breakdown diagrams, usable host calculations
- **Part 6 — Routing:** Static vs dynamic routing, routing protocols (RIP/OSPF/EIGRP/BGP) with metrics and AD values, administrative distance table, routing table codes, longest prefix match
- **Part 7 — Switching & VLANs:** Three switch functions (learning, forwarding, loop avoidance), VLANs (access vs trunk ports, 802.1Q, native VLAN, inter-VLAN routing, voice VLAN), STP (802.1D port states, root bridge, broadcast storms), RSTP (802.1w)
- **Part 8 — Wireless Networking:** 802.11 standards (a/b/g/n/ac/ax with Wi-Fi names), channel/frequency guidance, wireless security (WEP/WPA/WPA2/WPA3), wireless modes (ad-hoc/BSS/ESS), thick vs thin APs, IoT technologies (Z-Wave, Zigbee, ANT+, Bluetooth, NFC, RFID, IR)
- **Part 9 — Remote Access & VPNs:** VPN types (site-to-site, client-to-site, clientless), full vs split tunnel, VPN protocols (IPsec, SSL/TLS, PPTP, L2TP, OpenVPN), remote access protocols (RDP, VNC, SSH, Telnet)
- **Part 10 — Network Operations:** Performance thresholds (CPU, memory, NIC utilization), interface statistics (CRC errors, giants, runts, duplex mismatch), documentation types (physical/logical diagrams, baselines), SNMP (MIB, OIDs, traps), Syslog severity levels, NetFlow, change management
- **Part 11 — High Availability & DR:** NIC teaming, FHRP (HSRP/VRRP), active/active vs active/passive, DR site types (hot/warm/cold/cloud), RTO vs RPO, MTBF vs MTTR, UPS/PDU/generator/fire suppression
- **Part 12 — Cloud & Data Center:** Cloud deployment models (public/private/community/hybrid), cloud service models (IaaS/PaaS/SaaS), SDN (application/control/infrastructure layers), data center locations, Infrastructure as Code
- **Part 13 — Network Security:** Firewall types (stateful, NGFW, DMZ), IDS/IPS, proxy types, authentication protocols (RADIUS/TACACS+/Kerberos/LDAP/LDAPS), 802.1X (supplicant/authenticator/auth server), network attacks (DDoS, ARP spoofing, DNS poisoning, VLAN hopping, MAC flooding, rogue DHCP/AP)
- **Part 14 — Troubleshooting:** CompTIA 7-step methodology, sequential ping test (loopback → own IP → gateway → remote IP → hostname), full CLI diagnostic command reference (ping, tracert, pathping, nslookup, netstat, nmap, Wireshark, tcpdump), common issues quick reference
- **Part 15 — Ports Reference:** Complete port grid — all 33 exam-relevant ports with TCP/UDP designation
- **Part 16 — WAN Technologies:** MPLS (labels, LSPs, LSRs, LERs), leased lines, DSL, cable, fiber, cellular, satellite, fixed wireless, SD-WAN, Metro Ethernet, Frame Relay, CSU/DSU
